Case-specific checks

To specify any additional case-specific checks e:check tag may be used.

Examples

checks

checks
The following markup:
<e:get url="/mirror/headers">
  <e:case desc="Case 1">
    <e:expected> { "GET": "/mirror/headers", "Authorization": null, "Accept-Language": null, "cookies": {} } </e:expected>
    <e:check>
      <p> Some additional check for 
        <span cc:assertTrue="true">case 1</span>
      </p>
    </e:check>
  </e:case>
  <e:case desc="Case 2">
    <e:expected> { "GET": "/mirror/headers", "Authorization": null, "Accept-Language": null, "cookies": {} } </e:expected>
    <e:check>
      <p> Some additional check for 
        <span cc:assertTrue="true">case 2</span>
      </p>
    </e:check>
  </e:case>
</e:get>
